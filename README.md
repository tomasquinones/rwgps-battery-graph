# rwgps-battery-graph
Graphs battery usage, data from RWGPS device logs

## Requires: ##
pyperclip
`pip3 install pyperclip`

plotly express
`pip3 install plotly`

## Use: ##
1. View the log on the website (admins only)
2. Select all
3. Copy (now it's all in your clip board)
4. Run `python3 battery_graph.py`
